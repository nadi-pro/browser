# Getting Started

This section covers everything you need to start using the Nadi Browser SDK in your web application.

## Overview

The Nadi Browser SDK provides automatic monitoring for:

- **Core Web Vitals** - LCP, INP, CLS, FCP, TTFB
- **Session tracking** - User sessions and stability metrics
- **Error capture** - JavaScript errors and unhandled rejections
- **Breadcrumbs** - User action tracking for debugging

## Prerequisites

Before you begin, ensure you have:

- A Nadi account with an application configured
- Your application token and API key
- A web application using modern JavaScript (ES2020+)

## Browser Support

The SDK supports all modern browsers:

| Browser | Minimum Version |
|---------|-----------------|
| Chrome  | 64+             |
| Firefox | 67+             |
| Safari  | 12+             |
| Edge    | 79+             |
